POPEngine

JS library to create the pop-up windows

Open JavaScript library designed for creating pop-up windows, made by standard ECMAScript 5

Installing

  1. Download library here â‡©
  2. In head link css
    <head>  <link rel="stylesheet" href="POPEngine.min.css"> </head>
  3. At the end of page add js
    </body>  <script src="POPEngine.min.js"></script> </html>

Examples

  1. Creating element for future pop-up
    <div id="dpop"></div>;
  2. Initialize the object. The first option - id of element. Second - content. Third - the object with options.
    var pop = new POPEngine('dpop', 'My Content', {  activator: '#openpop',  header: 'My Header' });

  2. Initialize the object (template: "default", animation: "slide", direction: "right"):
    var pop = new POPEngine('dpop', 'Hello World', {  activator: '#openpop',  header: 'My Header',  animation: 'slide',  direction: 'right',  buttons: [  {    id: 'ok', title: 'OK', function: function () {     alert(1);     return false;    }  },   {id: 'cancel', title: 'Cancel'} ] });

Options

animation: 'fade'
Setting type of animation: (For default and notice: instant, slide, fade; sideblock: slide)
autoclose: '5s'
Set the time of autoclosing of notification in seconds or milliseconds
activator: 'cssselector'
Sets the items that will open pop-up on click
buttons: [{id:'ok',title:'OK',function:function(){alert(1);return false;}}]
Set the functions for buttons that will be performed on the click
class: 'addition clases'
Set addition classes
closer: 'cssselector'
Set addition elements for closing pop-up by click
cross: true
Set display cross for closing
direction: 'top'
Set the direction of appearance
duration: '1s'
Set duration of animation
ground: true
Set to display background closer
header: 'My Header'
Set header (if empty, the header does not appear)
height: '230px'
Set height (% or px)
left: 'center'
Set left indent, can take 'center',% or px
sideKeyThick: '28px'
Set width of "label" (template: sideblock)
top: 'center'
Set top intent, can take 'center',% or px
template: 'default'
Set template for pop-up, can take values: default, sideblock, notice
