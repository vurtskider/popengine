POPEngine

JS бiблiотека для створення вспливаючих вiкон

Відкрита JavaScript бібліотека, призначена для створення вспливаючих вікон, виконана за стандартом ECMAScript 5

Встановлення бiблiотеки

  1. Завантажити бібліотеку тут ⇩
  2. В блоці head додати css
    <head>  <link rel="stylesheet" href="POPEngine.min.css"> </head>
  3. В кінці документа додати js
    </body>  <script src="POPEngine.min.js"></script> </html>

Приклад використання

  1. Створюємо елемент з якого потім буде зроблено pop-up
    <div id="dpop"></div>;
  2. Проводимо ініціалізацію об'єкта. Перший параметр - id елемента. Другий - контент. Третій - об'єкт з опціями.
    var pop = new POPEngine('dpop', 'My Content', {  activator: '#openpop',  header: 'My Header' });

Додатковi приклади

  1. Ініціалізація об'єкта (шаблон:"default", анімація: "slide", напрямок: "top"):
    var pop = new POPEngine('dpop', 'Hello World', {  activator: '#openpop',  header: 'My Header',  animation: 'slide',  direction: 'top',  buttons: [  {    id: 'ok', title: 'OK', function: function () {     alert(1);     return false;    }  },   {id: 'cancel', title: 'Cancel'} ] });
  2. Ініціалізація об'єкта (шаблон:"default", анімація: "slide", напрямок: "right"):
    var pop = new POPEngine('dpop', 'Hello World', {  activator: '#openpop',  header: 'My Header',  animation: 'slide',  direction: 'right',  buttons: [  {    id: 'ok', title: 'OK', function: function () {     alert(1);     return false;    }  },   {id: 'cancel', title: 'Cancel'} ] });
  3. Ініціалізація об'єкта (шаблон:"default", анімація: "fade"):
    var pop = new POPEngine('dpop', 'Hello World', {  activator: '#openpop',  header: 'My Header',  animation: 'fade',  buttons: [  {    id: 'ok', title: 'OK', function: function () {     alert(1);     return false;    }  },   {id: 'cancel', title: 'Cancel'} ] });
  4. Ініціалізація об'єкта (шаблон:"sideblock", анімація: "slide", напрямок: "right"):
    var pop = new POPEngine('side-pop', 'Hello World', {  template: 'sideblock',  direction: 'right',  height: '70px',  width: '120px' });
  5. Ініціалізація об'єкта (шаблон:"notice", анімація: "fade"):
    var pop = new POPEngine('notice', 'Hello World', {  template: 'notice',  animation: 'fade',  activator: '#not',  height: '55px',  top: '4px',  autoclose: '2s' });

Параметри

animation: 'fade'
Встановлює тип анімації: (Для default і notice: instant, slide, fade; sideblock: slide)
autoclose: '5s'
Встановлює час автозакриття сповіщення (notice) у секундах або мілісекундах
activator: 'cssselector'
Встановлює елементи, клік на які відкриватиме pop-up
buttons: [{id:'ok',title:'OK',function:function(){alert(1);return false;}}]
Встановлює кнопки та функції, які будуть виконуватися на їх клік
class: 'addition clases'
Встановлює додаткові класи
closer: 'cssselector'
Встановлює додаткові елементи, клік на які закриватиме pop-up
cross: true
Встановлює чи відображати хрест для закриття
direction: 'top'
Встановлює напрямок появи
duration: '1s'
Встановлює тривалість анімації
ground: true
Встановлює чи відображати фонове затемнення
header: 'My Header'
Встановлює заговок (якщо порожній, то блок не відображається)
height: '230px'
Встановлює висоту (% або px)
left: 'center'
Встановлює відступ зліва, може приймати 'center', % або px
sideKeyThick: '28px'
Встановлює товщину "ярлика" (шаблон: sideblock)
top: 'center'
Встановлює відступ зверху, може приймати 'center', % або px
template: 'default'
Встановлює шаблон pop-up, може приймати значення: default, sideblock, notice
width: '350px'
Встановлює ширину (% або px)
